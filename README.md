# English Learning Website

An interactive vocabulary learning web application for students, featuring scientifically-proven memory techniques including spaced repetition (SM-2 algorithm), mnemonic tips, example sentences, and associative memory aids.

## Features

- **Scientific Memory Methods**: Uses SM-2 spaced repetition algorithm for optimal retention
- **Three Learning Modes**:
  - 📖 Memory Mode: Traditional flashcard learning with images
  - ✏️ Spelling Mode: Practice spelling by seeing translations
  - 🔄 Review Mode: Spaced repetition review
- **Rich Vocabulary Data**: Words organized by Longman Express 2nd Edition curriculum
- **Audio Pronunciation**: Text-to-speech support for English words
- **Progress Tracking**: LocalStorage-based progress saving
- **Import/Export**: Backup and restore your learning progress
- **Responsive Design**: Works on desktop and mobile devices

## Curriculum Structure

Based on Longman Express 2nd Edition:
- **Level 1** (1A, 1B): Basic vocabulary - self-introduction, family, school supplies, time, seasons, weather
- **Level 2** (2A, 2B): Daily life - rooms, furniture, city places, transportation, hobbies, sports
- **Level 3** (3A, 3B): Advanced expressions - appearance, personality, emotions, technology, education
- **Level 4** (4A, 4B): Deep topics - social issues, environment, travel, culture, friendship
- **Level 5** (5A, 5B): Advanced application - media influence, technology, environment, life philosophy

## Getting Started

Simply open `index.html` in a web browser to start learning.

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Web Speech API for pronunciation
- LocalStorage for data persistence
